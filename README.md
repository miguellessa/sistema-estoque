# Sistema de Estoque em Java

Projeto desenvolvido para praticar Programação Orientada a Objetos com Java.

## Conceitos aplicados
- Encapsulamento
- Herança
- Polimorfismo
- Classe abstrata
- Interface
- ArrayList
- Separação de responsabilidades com service

## Funcionalidades
- Cadastrar alimentos
- Cadastrar eletrônicos
- Listar produtos
- Mostrar preço final com desconto

## Estrutura
- `Produto`: classe abstrata base
- `Alimento` e `Eletronico`: subclasses
- `Vendavel`: interface
- `EstoqueService`: regras do sistema
- `App`: menu principal
